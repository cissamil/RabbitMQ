# RabbitMQ
 Escreva duas aplicações usando Spring Boot e RabbitMQ, usando Spring AMQP:

##Produtor: 

É um Webservice REST, que recebe uma mensagem de texto no corpo da mensagem via POST e a escreve em uma fila de mensagens do RabbitMQ

##Consumidor

Consome mensagens de texto da mesma fila de mensagens e escreve no console
